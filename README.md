<h1>Medição da margem de risco com base em fatores ambientais</h1> 

Controle sobre o descarte animal com base no princípio da Redução dos 3Rs: medição da margem de risco para auxiliar no cálculo da margem de segurança, 
de acordo com a linhagem e análise das variáveis ambientais.

Projeto foi desenvolvido em Flask.
